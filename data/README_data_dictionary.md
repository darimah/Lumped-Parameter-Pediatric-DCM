# Data Dictionary
- `HR` : heart rate (bpm)
- `l_lv`, `l_rv` : long-axis ventricle length (cm)
- `MAP` : mean arterial pressure (mmHg)
- `CO` : cardiac output (L/min)
- `Pao_sys`, `Pao_dias` : tekanan aorta sistolik/diastolik (mmHg)
- `V_lv_es`, `V_lv_ed`, `V_rv_es`, `V_rv_ed` : volume ventrikel (mL)
- `D_lv_es`, `D_lv_ed`, `D_rv_es`, `D_rv_ed` : diameter ventrikel (cm)
